# Finance Manager\n\n## Description\nThe Finance Manager is a personal finance management application that helps users manage their budgets, debts, and other financial aspects.\n\n## Features\n- User authentication\n- Budget tracking\n- Debt management\n- Email notifications\n\n## Installation\n1. Clone this repository: `git clone https://github.com/k9shuk/FinanceManager.git`\n2. Navigate to the project directory: `cd FinanceManager`\n3. Install dependencies: `npm install` (for both client and server)\n\n## Usage\n- Start the server: `npm run start`\n- Run the client in development mode: `npm run client`\n\n## Environment Variables\nMake sure to create a .env file based on the .env.example provided.\n\n## License\nThis project is licensed under the MIT License.
